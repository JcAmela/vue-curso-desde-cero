# Vue, paso a paso

Tutorial interactivo con **Vue 3**, **Vite**, **Vue Router** y **`@vue/repl`**. Un flujo oscuro centrado en el editor: pasos cortos con comprobación en el cliente; el progreso y el código se guardan en `localStorage` (sin cuenta ni servidor).

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run preview
```

Despliegue recomendado: [Vercel](https://vercel.com) (el repo incluye `vercel.json` con SPA fallback).
